Principais formatações:

Negrito / Destque

Nesta frase tem um termo em negrito usando a tag "<b>" (Não semântica)

Nesta frase tem um termo em negrito usando a tag "<strong>" (Semântica)


Itálico / Ênfase

Nesta frase tem um termo em itálico usando a tag "<i>" (Não semântica)

Nesta frase tem um termo em itálico usando a tag "<em>" (Semântica)


Pelo fato de o HTML5 ser uma lingagem semantica, é indicado usar as tags semanticas, neste caso "<strong>" e "<em>".
Os comandos "<b>" e "<i> são do HTML4 e que pode se tornar obsoletos e perder a função."


Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Podemos criar também um texto marcado usando um CSS3 pontual dentro da tag. Neste caso mudar a cor da marcação


Texto Grande e Pequeno:

Estamos criando um texto grande e outro texto pequeno neste parágrafo.


Texto Deletado

Podemeos marcar um texto como exluído para indicar que ele deve ser lido mas não cosniderado


Texto Inserido

Podemos marcar um texto como inserido para dar uma enfaze e indicar que ele foi inserido depois

Existe também o texto sublinhado, que usa a tag <u> (Não semântica)

Texto Subscrito
